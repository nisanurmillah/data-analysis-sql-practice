# JOIN Multi Tabel

Folder ini berisi query JOIN antara:
- customers
- orders
- orderdetails
- products
- suppliers
- categories

## Analisis yang Dilakukan
- Melihat relasi customer dengan produk yang dibeli
- Menampilkan detail order lengkap (produk, pengirim, tanggal)
- Mengetahui supplier dari produk yang dibeli customer

## Insight
JOIN digunakan untuk mengubah data terpisah menjadi informasi bisnis
yang utuh, seperti histori pembelian customer dan alur distribusi produk.
