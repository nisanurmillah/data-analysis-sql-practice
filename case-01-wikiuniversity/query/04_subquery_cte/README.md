# Subquery & CTE (WITH Clause)

Folder ini berisi query lanjutan menggunakan:
- Subquery di SELECT & WHERE
- IN, EXISTS
- Common Table Expression (CTE)

## Analisis yang Dilakukan
- Membandingkan penjualan produk terhadap total keseluruhan
- Menghitung kontribusi produk terhadap supplier
- Analisis persentase revenue

## Insight
CTE digunakan untuk meningkatkan keterbacaan query
dan mempermudah analisis bertingkat.
